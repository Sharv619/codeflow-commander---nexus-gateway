# Codeflow CLI

Generative Engineering Co-Pilot CLI for AI-powered development workflows.

## Features

- AI-assisted code generation and refactoring
- Intelligent code analysis and suggestions
- Multi-provider AI support (Gemini, OpenAI, Claude)
- Context-aware recommendations
- Integration with Enterprise Knowledge Graph

## Quick Start

```bash
npm install
npm run build
npm link
```

## Commands

```bash
npm run dev          # Start development
npm run build        # Build package
npm run test         # Run tests
npm run test:watch   # Watch mode
npm run test:coverage # Coverage report
npm run lint         # ESLint check
npm run lint:fix     # Auto-fix lint issues
npm run format       # Prettier formatting
npm run type-check   # TypeScript type checking
```

## Tech Stack

- TypeScript
- Commander.js
- Jest (testing)
- ESLint + Prettier
