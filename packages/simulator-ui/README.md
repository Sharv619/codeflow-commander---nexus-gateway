# Simulator UI

Interactive CI/CD pipeline simulator built with React 19 and Vite 7.

## Features

- Visual pipeline builder with drag-and-drop stages
- Live code analysis with ESLint and Jest integration
- AI-powered code review via Gemini/OpenAI/Claude
- Real-time deployment simulation
- GraphQL integration for enterprise knowledge graph data

## Quick Start

```bash
npm install
npm run dev
```

Access at: http://localhost:5173

## Tech Stack

- React 19
- Vite 7
- TailwindCSS
- Apollo Client (GraphQL)
- Jest (testing)

## Commands

```bash
npm run dev          # Start dev server (port 5173)
npm run build        # Production build
npm run test         # Run Jest tests
npm run test:watch   # Watch mode
npm run preview      # Preview production build
```
